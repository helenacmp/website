04.04.01 Temaopgave

Den endelige opgave på dette tema bestod af et interaktivt animeret spil, som vi selv skulle designe og programmere. Der var en række tekniske krav, som spillet skulle opfylde, bl.a. ift. spilstrukturen, UX-elementer mv. Udover selve spillet, skulle vi også aflevere et website, som skulle indeholde dokumentation for udviklingen af spillet. Det indebar bl.a. en beskrivelse af ideen, en teknisk dokumentation gennem diagrammer, skitser mv.

Introduktion til grundlæggende web

I tema to blev vi introduceret til de mest grundlæggende redskaber inden for multimediedesign. Først og fremmest blev vi introduceret til Visual Studio Code, og hvordan man sætter websider op i HTML og CSS og derefter uploader til sit eget domæne gennem FTP-programmet FileZilla. I website opgaven lærte jeg, hvordan man teknisk sætter en website op med forudbestemte wireframes og layoutdiagrammer. Det gjorde jeg bl.a. ved at lave grids og flexbox med bestemte id- og class-navne.

En vigtig ting jeg lærte, var også brug af media queries, som er en property man bruger i sit CSS-dokument. Media queries bruges til at lave et responsivt design, og det er helt afgørende, eftersom man som designer aldrig ved, hvilken platform ens side bliver vist på. Media queries gør, at vi kan styre, hvordan vores website skal ses på forskellige enheder.
